INSTRUCTIONS
  • You will be given a random number (Target Score) at the start of the game.
  • There are four crystals. By clicking on each crystal you will add specific amount of score to the total score.
  • You win the game by matching the total score to the target score.
  • You will lose the game if the total score goes over the target score
  • The value of each crystal is hidden until you click on it
  • Each time the game starts, the game will change the values of each crystal
